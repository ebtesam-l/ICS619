# Checkpoints

Fine-tuned weights for the four baselines are hosted on Google Drive:

**https://drive.google.com/drive/folders/1ZsEfyivmqn9ZEwPqPWW4UjSzWFeB9W0r?usp=share_link**

The folder contains four subfolders, one per baseline:

```
checkpoints/
├── gfpgan/
├── psfrgan/
├── codeformer/
└── difface/
```

Download all four and place them under `checkpoints/` so the layout above is reproduced locally.

For the IDS metric, also download the ArcFace `resnet18_110.pth`:
https://drive.google.com/drive/folders/1k3RCSliF6PsujCMIdCD1hNM63EozlDIZ
